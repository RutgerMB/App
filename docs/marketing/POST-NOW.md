**Display name:** `RepLock`

### Bio (paste)

```
Earn screen time with real workouts.
Apps stay locked until you move.
Search RepLock
```

### Profile picture

- Use **`assets/app-icon.png`** (or `assets/logo-source.png` / `public/icon-512.png`)
- Square crop on dark `#0A0A0B` if needed — readable at ~40px
- Match App Store icon so profile → store recognition is instant
- No face, no gym selfie, no purple glow chrome

### Link

- Pre-store: waitlist / landing / Linktree (“Coming soon on iOS”)
- Live: App Store URL + `?utm_source=tiktok&utm_medium=organic&utm_campaign=gymbro-w1`
- **Never** put the store URL in the caption body

## Upload checklist (every carousel day)

1. TikTok → **+** → **Photo** (not Video)
2. Select **`slide-01.png` → `slide-06.png`** in that order (6 images)
3. **Add sound** — trending **instrumental** phonk / gym edit (new sound every post)
4. Sync beat drop / build to **slide 4** (solution)
5. Paste **Caption** from the day below
6. Post → within **60s** paste **Pin comment**
7. Reply to comments for 30–60 min
