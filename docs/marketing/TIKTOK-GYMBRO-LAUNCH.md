## 3. Using slideshow packs 1‚Äì10 without looking spammy

Assets live in `docs/marketing/slides/slideshow-N/` (`slide-01` ‚Üí `slide-06`). Full upload checklist: [`POST-READY.md`](./POST-READY.md).

### Anti-spam rules

1. **Never dump all 10 in 10 days with identical caption structure.** Space carousels with demos and opinion clips.
2. **One product mention per caption** ‚Äî mechanic once, not a feature list.
3. **Rotate caption angles:** shame ‚Üí curiosity ‚Üí dry humor ‚Üí ‚Äúphone problem not program problem.‚Äù
4. **New sound on every post.** Same phonk track across five posts = spam signal.
5. **Repost winners only after 7+ days**, new sound + lightly edited hook if possible.
6. **Max 1‚Äì2 hard CTAs/week** in captions; other posts can end with a question.
7. **Don‚Äôt say ‚ÄúRepLocks.‚Äù** Brand is **RepLock**.

### Recommended pack ‚Üí angle mapping

| Pack | Theme | Best angle |
|------|-------|------------|
| 1 | Leg Day Excuses | Avoidance comedy ‚Üí lock until you train |
| 2 | Rest Timer Scrolling | Specific gym behavior (highest save potential) |
| 3 | No Gains Arc | Reframe: phone, not program |
| 4 | Gym Crush Cardio | Light humor ‚Äî don‚Äôt overplay the crush |
| 5 | Phone Spotter | Metaphor post ‚Äî strong pin candidate |
| 6 | Morning Scroll | Habit / morning routine niche crossover |
| 7 | Plateau Theory | Self-improvement TikTok crossover |
| 8 | Sunday Reset | Weekly ritual content |
| 9 | Push-Up Tax | Clearest mechanic explanation |
| 10 | Mirror vs Phone | Form-check irony |

## 5. Pin comment & sound strategy

### Pin comment (post within 60 seconds)

| Phase | Pin text |
|-------|----------|
| Pre-store | `Search RepLock ‚Äî earn screen time with real workouts` |
| Store live | `Link in bio ‚Äî free download. Earn minutes with push-ups, squats, planks.` |
| Engagement bait (alternate 1√ó/week) | `What‚Äôs your worst gym scroll habit? I‚Äôll reply.` |

Keep pins factual. One line + optional question. No ‚Äúlink in comments‚Äù spam.

### Sound strategy

| Rule | Detail |
|------|--------|
| **Always add sound** | Slideshows without audio underperform |
| **Prefer instrumental** | Phonk / gym edit / slowed ‚Äî lyrics fight on-screen text |
| **Sync** | Beat drop on **slide 4** (solution) for packs 1‚Äì10 ‚Äî see `POST-READY.md` audio table |
| **Rotate** | New sound every post; never reuse the same track more than once per week |
| **Reposts** | Same carousel + **new** sound = fresh distribution test |
| **Demos** | Native UI sounds optional; bed track low under VO |
