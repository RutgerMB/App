# RepLock TikTok — Gymbro Launch Package

**Status:** Post-ready · not started yet  
**Audience:** 18–30 men who train regularly and lose sessions to their phone  
**Tone:** Confident, dry humor, specific — not slang-stacked, not corporate, not childish  
**Companion assets:** [`POST-READY.md`](./POST-READY.md) · [`slides/slideshow-1`–`10`](./slides/) · [`CONTENT-CALENDAR.md`](./CONTENT-CALENDAR.md)

This is the **profile + script system** for standing up TikTok from zero. Use `POST-READY.md` for upload checklists and regenerated PNGs.

## 1. Profile setup

### Username ideas (pick one; claim aliases if free)

| Priority | Handle | Why |
|----------|--------|-----|

| 2 | `@getreplock` | Clear action if brand is taken |
| 3 | `@replockapp` | Platform clarity for cold traffic |
| 4 | `@earnwithreplock` | Mechanic in the name (secondary) |

**Display name:** `RepLock` (not “RepLock App 💪”). Keep it store-clean.

### Bio options (pick one)

**A — Mechanic (recommended pre-launch)**  
```
Earn screen time with real workouts.
Apps stay locked until you move.
Search RepLock
```

**B — Problem → product**  
```
Doomscroll tax, paid in push-ups.
Lock the apps. Earn the minutes.
```

**C — Gym-specific**  
```
Rest timer ≠ scroll timer.
Reps unlock your feed.
```

Keep bios under 3 lines. No emoji walls. Update line 3 when the store link is live (see link strategy below).

### Profile picture

- Use the **RepLock wordmark or app icon** on `#0A0A0B` — not a face, not a gym selfie.
- Square crop, high contrast, readable at ~40px (TikTok avatar size).
- Match App Store icon so profile → store recognition is instant.
- Avoid purple glow / neon chrome; stay premium dark.

### Link-in-bio strategy

| Phase | Bio link | Pin comment CTA |
|-------|----------|-----------------|
| **Pre–App Store** | Waitlist / landing / Linktree with email capture + “Coming soon on iOS” | `Search RepLock on the App Store when we drop` |
| **Store live** | Direct App Store URL (or Linktree: iOS + Android) with UTM | `Link in bio — free download` |
| **UTM template** | `?utm_source=tiktok&utm_medium=organic&utm_campaign=gymbro-w1` | Rotate `campaign` per week (`gymbro-w2`, etc.) |

**Rules:** Never put the store URL in the caption body (hurts reach). Drive profile visits → bio. One link only.

| **1. Gym phone problem** | ~40% | Relatable failure: rest-timer scroll, skipped legs, morning scroll | Slideshow packs 1–10 |

| Week | Volume | Mix |
|------|--------|-----|

**Reply window:** First 60 minutes — answer every comment like a human, not a brand bot.

## 3. Using slideshow packs 1–10 without looking spammy

Assets live in `docs/marketing/slides/slideshow-N/` (`slide-01` → `slide-06`). Full upload checklist: [`POST-READY.md`](./POST-READY.md).

### Anti-spam rules

1. **Never dump all 10 in 10 days with identical caption structure.** Space carousels with demos and opinion clips.
2. **One product mention per caption** — mechanic once, not a feature list.
3. **Rotate caption angles:** shame → curiosity → dry humor → “phone problem not program problem.”
4. **New sound on every post.** Same phonk track across five posts = spam signal.
5. **Repost winners only after 7+ days**, new sound + lightly edited hook if possible.
6. **Max 1–2 hard CTAs/week** in captions; other posts can end with a question.
7. **Don’t say “RepLocks.”** Brand is **RepLock**.

### Recommended pack → angle mapping

| Pack | Theme | Best angle |
|------|-------|------------|
| 1 | Leg Day Excuses | Avoidance comedy → lock until you train |
| 2 | Rest Timer Scrolling | Specific gym behavior (highest save potential) |
| 3 | No Gains Arc | Reframe: phone, not program |
| 4 | Gym Crush Cardio | Light humor — don’t overplay the crush |
| 5 | Phone Spotter | Metaphor post — strong pin candidate |
| 6 | Morning Scroll | Habit / morning routine niche crossover |
| 7 | Plateau Theory | Self-improvement TikTok crossover |
| 8 | Sunday Reset | Weekly ritual content |
| 9 | Push-Up Tax | Clearest mechanic explanation |
| 10 | Mirror vs Phone | Form-check irony |

## 5. Pin comment & sound strategy

### Pin comment (post within 60 seconds)

| Phase | Pin text |
|-------|----------|
| Pre-store | `Search RepLock — earn screen time with real workouts` |
| Store live | `Link in bio — free download. Earn minutes with push-ups, squats, planks.` |
| Engagement bait (alternate 1×/week) | `What’s your worst gym scroll habit? I’ll reply.` |

Keep pins factual. One line + optional question. No “link in comments” spam.

### Sound strategy

| Rule | Detail |
|------|--------|
| **Always add sound** | Slideshows without audio underperform |
| **Prefer instrumental** | Phonk / gym edit / slowed — lyrics fight on-screen text |
| **Sync** | Beat drop on **slide 4** (solution) for packs 1–10 — see `POST-READY.md` audio table |
| **Rotate** | New sound every post; never reuse the same track more than once per week |
| **Reposts** | Same carousel + **new** sound = fresh distribution test |
| **Demos** | Native UI sounds optional; bed track low under VO |

## 6. First 7 days — day-by-day plan

Assume profile live, bio set, packs generated (`npm run marketing:slides`).

| Day | Post | Script | Asset | Goal |
|-----|------|--------|-------|------|
| **1** | Slideshow | 01 Rest timer | `slideshow-2/` | Highest-relatability opener |
| **2** | Slideshow | 03 Phone spotter | `slideshow-5/` | Metaphor + brand memory |
| **3** | Screen demo | 08 Product demo | In-app record | Prove the mechanic |
| **4** | Slideshow | 02 Leg day | `slideshow-1/` | Humor without slang overload |
| **5** | Opinion | 09 Discipline take | B-roll | Break carousel pattern |
| **6** | Slideshow | 05 Push-up tax | `slideshow-9/` | Clearest earn-minutes explain |
| **7** | Slideshow or founder | 04 No gains **or** 12 Founder | `slideshow-3/` or talking head | Soft proof / trust |

### Day-of checklist

1. Upload 6 PNGs in order (or film script)  
2. Attach trending **instrumental** sound; sync drop to slide 4  
3. Paste caption from script (edit lightly to sound human)  
4. Post → pin comment within 60s  
5. Reply to comments for 30–60 min  
6. Log: views, completion rate, profile visits, saves  

### Days 8–14 (preview)

Use Scripts 06, 07, 10, 11, 13 + Script 14 challenge + **repost Day 1 or 2 winner with new sound**. Full 30-day map: [`CONTENT-CALENDAR.md`](./CONTENT-CALENDAR.md).
